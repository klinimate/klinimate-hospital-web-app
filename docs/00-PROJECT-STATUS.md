Foundation Version: 2.0

The Klinimate Product Foundation (Vision, PRD, Clinical Workflow, Database Design, UI/UX, Clinical Design System, Data Architecture, Development Rules, and related core documents) is frozen.

Future changes should follow semantic versioning:

- Major Version (3.0): Structural or architectural changes.
- Minor Version (2.x): New features and modules.
- Patch Version (2.0.x): Documentation corrections and bug fixes.

Development should now prioritize implementation over redesign.
