# Klinimate Command Centre

Version: 1.0

Status: Active

## Purpose

The Klinimate Command Centre is the central clinical dashboard used by Klinimate Intensivists to remotely monitor and support multiple partner hospitals from one location.

The objective is to detect patient deterioration early, prioritize critically ill patients, and provide timely specialist recommendations.

The Command Centre should function like an Air Traffic Control system for hospitals.

---

## Users

Only:
- Klinimate Intensivist

---

## Primary Objectives

- View all assigned hospitals
- View all admitted patients
- Prioritize critical patients
- Detect clinical deterioration early
- Review Klinimate AI Clinical Assessment
- Provide consultant recommendations
- Monitor patient progress remotely

---

## Dashboard Layout

Top Summary

- Total Hospitals
- Total Active Patients
- Green Patients
- Yellow Patients
- Red Patients

---

Hospital Cards

Each hospital should display

- Hospital Name
- Active Patients
- Green Count
- Yellow Count
- Red Count
- Last Update Time

---

Patient List

Display

- Name
- Age / Sex
- UHID
- Ward / ICU
- Bed Number
- Primary Diagnosis
- Triage Colour
- Last Updated
- Assigned Hospital Medical Officer

---

Patient Priority

Always display

1. Red
2. Yellow
3. Green

Critical patients must always appear first.

---

Patient Details

One tap should open

- Admission Summary
- Daily Progress Round
- Vitals Timeline
- Investigations
- Medications
- Klinimate AI Clinical Assessment
- Consultant Recommendations

---

Quick Actions

- Review Patient
- Add Recommendation
- Escalate
- Call Hospital
- Start Video Consultation
- View Previous Notes

---

Notifications

Receive alerts when

- New Red Patient
- New Yellow Patient
- Patient Deterioration
- Patient Improvement
- New Hospital Admission

---

Design Principles

- Mobile-first
- Minimal scrolling
- Fast loading
- One-hand operation
- WhatsApp-like simplicity
