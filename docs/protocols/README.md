# Klinimate Clinical Protocol Library

Version: 1.0

Status: Active

## Purpose

The Klinimate Clinical Protocol Library is the standardized medical knowledge base that powers Klinimate Intelligence and supports healthcare professionals across all care settings.

Protocols are evidence-informed, regularly updated, and designed to assist—not replace—clinical judgement.

## Standard Protocol Template

Every protocol follows the same Klinimate structure:

1. Overview
2. Clinical Presentation
3. Risk Factors
4. Clinical Assessment
5. Red Flags
6. Recommended Investigations
7. Klinimate Intelligence
8. Initial Management
9. Monitoring
10. Specialist Consultation
11. Escalation Criteria
12. Disposition
13. Follow-up
14. References

---

## Guiding Principle

Every protocol should simplify clinical decision-making while maintaining clinician autonomy and patient safety.
